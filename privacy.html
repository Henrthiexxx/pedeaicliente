Esta Politica de Privacidade explica como o Pedra Delivery coleta, usa, armazena e protege dados relacionados ao uso do aplicativo de delivery.

Dados que podemos coletar

  • Dados de conta, como nome, e-mail, telefone e identificador de usuario.
  • Dados de entrega, como endereco, bairro, complemento, referencia e localizacao GPS quando o usuario optar por capturar a localizacao.
  • Dados de pedidos, incluindo loja, produtos, valores, forma de pagamento selecionada, status e historico do pedido.
  • Dados de notificacoes, como tokens de push usados para avisar sobre status do pedido e comunicacoes do app.
  • Conteudo enviado pelo usuario, como observacoes do pedido, avaliacoes e comentarios.
  • Dados tecnicos locais, como caches de carrinho, notificacoes e preferencias de tema.

Como usamos os dados

  • Para criar e autenticar contas de usuario.
  • Para processar pedidos, calcular taxas, direcionar entregas e exibir o historico.
  • Para permitir que lojas e entregadores executem o pedido.
  • Para enviar notificacoes sobre pedidos, entregas, suporte e comunicacoes operacionais.
  • Para melhorar seguranca, prevenir abuso e dar suporte ao usuario.

Compartilhamento

Podemos compartilhar dados necessarios com lojas, entregadores e servicos tecnicos usados para operar o app, como Firebase/Google Cloud. O compartilhamento e limitado ao necessario para funcionamento, suporte, seguranca e cumprimento de obrigacoes legais.

Localizacao

A localizacao pode ser usada para facilitar o cadastro do endereco e a entrega do pedido. A captura depende da permissao do usuario no dispositivo. O usuario tambem pode preencher o endereco manualmente.

Retencao e exclusao

Os dados sao mantidos pelo tempo necessario para operar pedidos, suporte, seguranca, contabilidade e obrigacoes legais. O usuario pode solicitar a exclusao da conta e dos dados associados pela pagina de exclusao de conta.

Seguranca

Usamos servicos com conexao criptografada e controles de acesso para reduzir risco de acesso indevido. Nenhum sistema e totalmente imune a falhas, por isso tambem recomendamos que o usuario proteja sua senha e dispositivo.

Alteracoes nesta politica

Esta politica pode ser atualizada para refletir mudancas no app, na operacao ou em requisitos legais. A data de atualizacao sera ajustada quando houver alteracoes relevantes.
